## General

### Global Shortcuts

Default Browser

&emsp;Super + B

Default File Manager

&emsp;Super + F

Default Text Editor

&emsp;Super + T

Default Terminal

&emsp;Super + Enter

</br>

### General DE/WM config

Window Focus Navigation :: 

&emsp;Super + Left | Right | Up | Down 

Window Movement ::

&emsp;Super + Control + Left | Right | Up | Down 

Workspace Navigation

&emsp;Super + Page_Up | Page_Down 

Move Window to Workspace

&emsp;Super + Control + Page_Up | Page_Down 

Move Window to Monitor

&emsp;Super + Control + Comma[<] | Period [>]

Resize Window

&emsp;Super + J ; Decrease window height

&emsp;Super + I ; Decrease window width

&emsp;Super + K ; Increase window height

&emsp;Super + L ; Increase window width

</br>

### Niri-speciic Controls

Monitor Focus Navigation

&emsp;Super + Shift + Left | Right

Move Window to Monitor

&emsp;Super + Shift + Control + Left | Right

Consume or Expel Window ; lower window in the column by default

&emsp;Super + Comma[<] | Period [>]
